# Book2Skill Factory v2.1 - Critical Path Fix

## 🐛 Problem

When deploying plugins generated by Book2Skill Factory v2.0, users encountered the following error:

```
❌ Plugin directory not found at path:
   C:\Users\USERNAME\.claude\plugins\marketplaces\marketplace-name\skills\skill-name
```

**Impact:** All 7 skills in minimalist-entrepreneur-pro failed to load.

---

## 🔍 Root Cause

The `source` paths in `marketplace.json` were incorrect:

### ❌ Incorrect (v2.0)
```json
{
  "source": "./skills/scrum-framework",
  "source": "./skills/mvp-builder",
  ...
}
```

### ✅ Correct (v2.1)
```json
{
  "source": "./plugins/minimalist-entrepreneur-pro/skills/scrum-framework",
  "source": "./plugins/minimalist-entrepreneur-pro/skills/mvp-builder",
  ...
}
```

---

## 📂 Why This Happened

### Official Plugin Structure

```
minimalist-entrepreneur-pro-marketplace/
├── .claude-plugin/
│   └── marketplace.json          ← Config file here
└── plugins/
    └── minimalist-entrepreneur-pro/
        └── skills/
            ├── scrum-framework/
            ├── mvp-builder/
            └── ...
```

### Path Resolution

The `source` path in `marketplace.json` is **relative to the marketplace.json file itself**.

- **marketplace.json location:** `.claude-plugin/marketplace.json`
- **skills location:** `plugins/minimalist-entrepreneur-pro/skills/`
- **Relative path:** `./plugins/minimalist-entrepreneur-pro/skills/[skill-name]`

---

## ✅ What Was Fixed

### 1. Updated marketplace.json.template

**File:** `deploy/marketplace.json.template`

```diff
- "source": "./skills/tool-1",
+ "source": "./plugins/[plugin-name]/skills/tool-1",
```

### 2. Updated SKILL.md Documentation

**File:** `SKILL.md`

- Added CRITICAL warning about path structure
- Updated plugin structure diagram
- Added example with correct paths

### 3. Updated changelog.md

**File:** `changelog.md`

```markdown
## v2.1 (2026-03-26)
- 🐛 CRITICAL FIX: Corrected marketplace.json path configuration
  - Changed: `"source": "./skills/xxx"` → `"source": "./plugins/[plugin-name]/skills/xxx"`
```

### Files to Update in Existing Plugins
If you have existing plugins generated with v2.0, update their `marketplace.json`:

```bash
# Find all marketplace.json files
find ~/.claude/plugins/marketplaces -name "marketplace.json"

# Update source paths
# From: "source": "./skills/xxx"
# To:   "source": "./plugins/[plugin-name]/skills/xxx"
```

---

## 🚀 Deployment

### For New Plugins

Use the updated templates in v2.1 - paths will be correct automatically.

### For Existing Plugins

1. Locate the marketplace.json file
2. Update all `source` paths
3. Restart Claude Code

**Example update:**
```diff
{
  "plugins": [
    {
      "name": "my-tool",
-     "source": "./skills/my-tool"
+     "source": "./plugins/my-plugin/skills/my-tool"
    }
  ]
}
```
